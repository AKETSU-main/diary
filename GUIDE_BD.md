# 📘 РУКОВОДСТВО ПО РУЧНОМУ ИЗМЕНЕНИЮ БАЗЫ ДАННЫХ (schedule.json)

> **ВНИМАНИЕ:** Это руководство ТОЛЬКО для ручного редактирования файла `schedule.json`. Никаких скриптов, только текст.

---

## 📁 1. ОТКРЫТИЕ ФАЙЛА

1. Открой файл `schedule.json` в любом текстовом редакторе (Notepad++, VS Code, Sublime Text, обычный Блокнот).
2. **Сделай копию файла** перед правками, во избежание утери данных (назови `schedule_backup.json`).

---

## 🏗️ 2. СТРУКТУРА ФАЙЛА

Файл состоит из ТРЁХ главных разделов:

```json
{
  "classes": { ... },    // Расписание классов (какие уроки у класса)
  "teachers": { ... },   // Расписание учителей (где и кто преподаёт)
  "bells": { ... }       // Расписание звонков (время уроков)
}
```

---

## 🔔 3. ИЗМЕНЕНИЕ РАСПИСАНИЯ ЗВОНКОВ (`bells`)

Здесь хранится время начала и конца каждого урока.

### Где находится:
В самом низу файла, раздел `"bells"`.

### Структура:
```json
"bells": {
  "Понедельник": ["08:00-08:45", "08:50-09:35", "09:40-10:25", "10:30-11:15", "11:20-12:05"],
  "Вторник": ["08:00-08:45", "08:50-09:35", "09:40-10:25", "10:30-11:15", "11:20-12:05"],
  ...
}
```

### Как изменить время урока:
1. Найди нужный день (например, `"Понедельник"`).
2. В списке найди нужный урок по порядку:
   - 1-й элемент → 1 урок
   - 2-й элемент → 2 урок
   - и т.д.
3. Замени время в кавычках.

#### ✅ ПРИМЕР: Изменить 5-й урок на Понедельник с 11:20-12:05 на 11:30-12:15

**БЫЛО:**
```json
"Понедельник": ["08:00-08:45", "08:50-09:35", "09:40-10:25", "10:30-11:15", "11:20-12:05"]
```

**СТАЛО:**
```json
"Понедельник": ["08:00-08:45", "08:50-09:35", "09:40-10:25", "10:30-11:15", "11:30-12:15"]
```

⚠️ **ВАЖНО:**
- Меняй время ВО ВСЕХ днях, если расписание звонков одинаковое.
- Формат строго: `"ЧЧ:ММ-ЧЧ:ММ"` (кавычки обязательны).
- Не удаляй запятые между элементами!

---

## 🏫 4. ДОБАВЛЕНИЕ/ИЗМЕНЕНИЕ УРОКА У КЛАССА (`classes`)

Здесь хранится расписание каждого класса: предмет, учитель, кабинет, время.

### Где находится:
Раздел `"classes"`, внутри него название класса (например, `"5А"` или `"9Б"`).

### Структура одного урока:
```json
{
  "lesson": 1,
  "time": "08:00-08:45",
  "subject": "Математика",
  "room": "101",
  "teacher": "Иванова"
}
```

### Поля урока:
| Поле | Что значит | Пример |
|------|------------|--------|
| `lesson` | Номер урока (1-5) | `1` |
| `time` | Время (должно совпадать с `bells`) | `"08:00-08:45"` |
| `subject` | Предмет | `"Математика"` |
| `room` | Кабинет | `"101"` |
| `teacher` | Фамилия учителя (без инициалов) | `"Иванова"` |

**БЫЛО (во Вторник у 5А):**

### ✅ ПРИМЕР 2: Добавить 6-й урок у 5А в Понедельник

**Задача:** Добавить новый урок после 5-го.

1. Найди `"5А"` → `"Понедельник"`.
2. После последнего урока (5-го) поставь запятую.
3. Вставь новый блок:

**БЫЛО:**
```json
"Понедельник": [
  {...}, {...}, {...}, {...},
  {"lesson": 5, "time": "11:20-12:05", "subject": "Английский язык", "room": "204", "teacher": "Попова"}
]
```

**СТАЛО:**
```json
"Понедельник": [
  {...}, {...}, {...}, {...},
  {"lesson": 5, "time": "11:20-12:05", "subject": "Английский язык", "room": "204", "teacher": "Попова"},
  {"lesson": 6, "time": "12:10-12:55", "subject": "Факультатив", "room": "101", "teacher": "Иванова"}

```

⚠️ **ВАЖНО:**
- Не забудь добавить запятую после предыдущего урока!
- Если добавляешь 6-й урок, сначала добавь 6-е время в `"bells"`!

## 👨‍🏫 5. ДОБАВЛЕНИЕ/ИЗМЕНЕНИЕ УРОКА У УЧИТЕЛЯ (`teachers`)

Здесь хранится расписание учителей: у какого класса, какой предмет, где и когда.

### Где находится:
Раздел `"teachers"`, внутри него фамилия учителя (например, `"Иванова"`).

### Структура одного урока учителя:
```json
{
  "lesson": 1,
  "time": "08:00-08:45",
  "class": "5А",
  "subject": "Математика",
  "room": "101"
}
```

### Поля урока учителя:
| Поле | Что значит | Пример |
|------|------------|--------|
| `lesson` | Номер урока (1-5) | `1` |
| `time` | Время | `"08:00-08:45"` |
| `class` | Класс | `"5А"` |
| `subject` | Предмет | `"Математика"` |
| `room` | Кабинет | `"101"` |

⚠️ **ОТЛИЧИЕ от `classes`:** здесь поле называется `"class"`, а не `"teacher"`!

---

### ✅ ПРИМЕР: Добавить урок учителю Ивановой во Вторник

**Задача:** У Ивановой во Вторник сейчас 2 урока. Добавить 3-й.

1. Найди `"teachers"` → `"Иванова"` → `"Вторник"`.
2. Внутри списка после последнего элемента поставь запятую.
3. Вставь новый урок:

**БЫЛО:**
```json
"Иванова": {
  "Вторник": [
    {"lesson": 2, "time": "08:50-09:35", "class": "5А", "subject": "Математика", "room": "101"},
    {"lesson": 1, "time": "08:00-08:45", "class": "9Б", "subject": "Геометрия", "room": "101"}
  ]
}
```

**СТАЛО:**
```json
"Иванова": {
  "Вторник": [
    {"lesson": 2, "time": "08:50-09:35", "class": "5А", "subject": "Математика", "room": "101"},
    {"lesson": 1, "time": "08:00-08:45", "class": "9Б", "subject": "Геометрия", "room": "101"},
    {"lesson": 3, "time": "09:40-10:25", "class": "10А", "subject": "Алгебра", "room": "101"}
  ]
}
```

## 🔄 6. СИХРОНИЗАЦИЯ: КЛАСС ↔ УЧИТЕЛЬ

**ГЛАВНОЕ ПРАВИЛО:** Если ты меняешь расписание в `classes`, ты ОБЯЗАН обновить `teachers`!

### Алгоритм:
1. Ты изменил урок у класса **5А** (например, поставил учителя **Сидоров** вместо **Иванова**).
2. Найди в `teachers` старого учителя (**Иванова**) → удали этот урок из его расписания.
3. Найди нового учителя (**Сидоров**) → добавь этот урок в его расписание.

---

### ✅ ПОЛНЫЙ ПРИМЕР синхронизации

**Задача:** У 5А в Среду на 3-м уроке был "Английский (Попова)". Меняем на "Физкультура (Сидоров)".

#### ШАГ 1: Меняем в `classes`

Найди: `"classes"` → `"5А"` → `"Среда"` → 3-й урок.

**БЫЛО:**
```json
{"lesson": 3, "time": "09:40-10:25", "subject": "Английский язык", "room": "204", "teacher": "Попова"}
```

**СТАЛО:**
```json
{"lesson": 3, "time": "09:40-10:25", "subject": "Физкультура", "room": "Спортзал", "teacher": "Сидоров"}
```

#### ШАГ 2: Удаляем у старого учителя (Попова)

Найди: `"teachers"` → `"Попова"` → `"Среда"`.

**БЫЛО:**
```json
"Среда": [
  {"lesson": 3, "time": "09:40-10:25", "class": "5А", "subject": "Английский язык", "room": "204"}
]
```

**СТАЛО (удалили весь урок):**
```json
"Среда": []
```

#### ШАГ 3: Добавляем новому учителю (Сидоров)

Найди: `"teachers"` → `"Сидоров"` → `"Среда"`.

**БЫЛО:**
```json
"Среда": []
```

**СТАЛО:**
```json
"Среда": [
  {"lesson": 3, "time": "09:40-10:25", "class": "5А", "subject": "Физкультура", "room": "Спортзал"}
]
```

✅ Готово! Теперь данные синхронизированы.

**Задача:** Добавить класс **6В**.

### ШАГ 1: Добавь класс в `classes`

В разделе `"classes"` добавь новую запись:

```json
"6В": {
  "Понедельник": [],
  "Вторник": [],
  "Среда": [],
  "Четверг": [],
  "Пятница": []
}
```

### ШАГ 2: Заполни уроки

Для каждого дня добавь уроки (как в примере с 5А):

```json
"6В": {
  "Понедельник": [
    {"lesson": 1, "time": "08:00-08:45", "subject": "Математика", "room": "101", "teacher": "Иванова"},
    {"lesson": 2, "time": "08:50-09:35", "subject": "Русский язык", "room": "202", "teacher": "Петрова"},
    ...
  ],
  "Вторник": [...],
  ...
}
```

### ШАГ 3: Добавь учителей в `teachers`

Для каждого урока из ШАГА 2:
1. Найди учителя в `"teachers"`.
2. Добавь урок в соответствующий день.

Пример для учителя Ивановой (если у неё 1-й урок в Понедельник с 6В):

```json
"Иванова": {

    {"lesson": 1, "time": "08:00-08:45", "class": "6В", "subject": "Математика", "room": "101"}
  ]

```

## ❌ 9. ЧАСТЫЕ ОШИБКИ

| Ошибка | Как выглядит | Как исправить |
|--------|--------------|---------------|
| **Забытая запятая** | ` {...} {...} ` | Поставь запятую между объектами: `{...}, {...}` |
| **Лишняя запятая** | `[ {...}, ] ` | Убери запятую после последнего элемента |
| **Нет кавычек** | `subject: Математика` | Добавь кавычки: `"subject": "Математика"` |
| **Неверное поле** | `"teacher": "5А"` в teachers | В `teachers` должно быть `"class": "5А"` |
| **Расхождение времени** | В classes `08:00`, в bells `08:05` | Приведи время к одному значению |
| **Дубликат урока** | Два 3-х урока у учителя в один день | Проверь, не назначен ли учитель дважды |

## ✅ 10. ПРОВЕРКА ПОСЛЕ ПРАВОК

Перед тем как запускать бота:

1. **Проверь запятые:** После каждого `{...}` кроме последнего должна быть запятая.
2. **Проверь кавычки:** Все ключи и строковые значения в кавычках `""`.
3. **Проверь скобки:**
   - `{` закрывается `}`
   - `[` закрывается `]`
4. **Синхронизация:** Если урок есть в `classes`, он должен быть в `teachers` у указанного учителя.

### Быстрая проверка валидности JSON:

Если у тебя есть Python, выполни в терминале:
```bash
python -c "import json; json.load(open('schedule.json')); print('✅ JSON валиден!')"
```

Если ошибки нет → файл корректен.

## 🎯 12. ПРИМЕР ПОЛНОГО ИЗМЕНЕНИЯ

**Задача:** В Пятницу у 9Б на 5-м уроке был "Классный час (Иванова)". Меняем на "Информатика (Сидоров)" в кабинете "Комп. класс".

### Выполняем по шагам:

#### 1. Меняем в `classes` → `9Б` → `Пятница`:

```json
// БЫЛО:
{"lesson": 5, "time": "11:20-12:05", "subject": "Классный час", "room": "101", "teacher": "Иванова"}

// СТАЛО:
{"lesson": 5, "time": "11:20-12:05", "subject": "Информатика", "room": "Комп. класс", "teacher": "Сидоров"}
```

#### 2. Удаляем у `teachers` → `Иванова` → `Пятница`:

Находим урок с `"class": "9Б"` и `lesson: 5`, удаляем его из списка.

#### 3. Добавляем `teachers` → `Сидоров` → `Пятница`:

```json
// БЫЛО:
"Пятница": []

// СТАЛО:
"Пятница": [
  {"lesson": 5, "time": "11:20-12:05", "class": "9Б", "subject": "Информатика", "room": "Комп. класс"}
]
```

✅ Готово!
