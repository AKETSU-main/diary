# School Schedule Bot

Бот для просмотра расписания уроков и звонков на базе **telebot**.

## 🚀 Быстрый старт

### 1. Установка Python
Убедитесь, что у вас установлен Python 3.8 или выше:
```bash
python --version
```

### 2. Установка библиотеки telebot
```bash
pip install pyTelegramBotAPI
```

### 3. Настройка токена
Создайте файл `token.txt` в папке с проектом и вставьте туда токен вашего бота (получить у @BotFather в Telegram).

### 4. Запуск бота
```bash
python bot.py
```

## 📝 Как добавлять данные (schedule.json)

Файл `schedule.json` — это база данных. Всё редактируется вручную в этом файле.

### Структура файла:
```json
{
  "bells": { ... },
  "classes": { ... },
  "teachers": { ... }
}
```

---

### 1️⃣ Расписание звонков (`bells`)

Формат: для каждого дня недели — список времени уроков.

**Пример:**
```json
"bells": {
  "Понедельник": ["08:00 - 08:45", "08:50 - 09:35"],
  "Вторник": ["08:00 - 08:45", "08:50 - 09:35"]
}
```

**Как добавить:**
- Просто добавьте новый день (например, `"Среда": [...]`)
- Или добавьте новый урок в существующий день: `["08:00 - 08:45", "08:50 - 09:35", "09:45 - 10:30"]`

---

### 2️⃣ Классы и уроки (`classes`)

Формат: название класса → дни недели → список уроков.

**Пример урока:**
```json
{"lesson": 1, "time": "08:00 - 08:45", "subject": "Математика", "room": "101", "teacher": "Иванова М.И."}
```

**Как добавить класс:**
1. Откройте раздел `"classes"`
2. Добавьте новый блок с названием класса (например, `"5А"`)
3. Скопируйте структуру дней из другого класса
4. Замените предметы, учителей и кабинеты

**Пример добавления класса 5А:**
```json
"5А": {
  "Понедельник": [
    {"lesson": 1, "time": "08:00 - 08:45", "subject": "Математика", "room": "201", "teacher": "Иванова М.И."},
    {"lesson": 2, "time": "08:50 - 09:35", "subject": "Русский язык", "room": "201", "teacher": "Петрова А.С."}
  ],
  "Вторник": [ ... ]
}
```

⚠️ **Важно:**
- Название учителя должно точно совпадать с именем в разделе `teachers`
- Номер урока (`lesson`) должен соответствовать времени из `bells`

---

### 3️⃣ Учителя и их расписание (`teachers`)

Формат: ФИО учителя → дни недели → список уроков.

**Пример урока учителя:**
```json
{"lesson": 1, "time": "08:00 - 08:45", "class": "5А", "subject": "Математика", "room": "101"}
```

**Как добавить учителя:**
1. Откройте раздел `"teachers"`
2. Добавьте новый блок с ФИО (например, `"Сидоров В.В."`)
3. Укажите дни и уроки

**Пример:**
```json
"Сидоров В.В.": {
  "Понедельник": [
    {"lesson": 1, "time": "08:00 - 08:45", "class": "5А", "subject": "Физкультура", "room": "Спортзал"},
    {"lesson": 2, "time": "08:50 - 09:35", "class": "5Б", "subject": "Физкультура", "room": "Спортзал"}
  ],
  "Вторник": []
}
```

⚠️ **Важно:**
- Если у учителя нет уроков в день, оставьте пустой список: `"Понедельник": []`
- Данные должны совпадать с расписанием классов!

## ❗ Частые ошибки

| Ошибка | Решение |
|--------|---------|
| Бот не запускается | Проверьте, что `token.txt` существует и содержит только токен |
| Расписание не отображается | Проверьте JSON на ошибки (запятые, кавычки) на сайте jsonlint.com |
| Неверное время | Формат строго `"08:00 - 08:45"` (через пробел и тире) |
| Учитель не найден | ФИО должно точно совпадать в разделах `classes` и `teachers` |

## 💡 Советы

1. **Копируйте готовые блоки** — проще скопировать урок/класс и изменить данные, чем писать с нуля
2. **Проверяйте JSON** — используйте онлайн-валидаторы перед сохранением
3. **Делайте резервные копии** — перед массовыми изменениями сохраните копию файла
